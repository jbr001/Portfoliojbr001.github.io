How To Define a Table

How to define a table?

    There are six key components to defining an HTML table: Table Headers, Table Rows, Table Cells, grouping of the header content, grouping of the footer content, and Table Captions. To create a table, you use the tag <table>.A table header is denoted by <th> and it is used to define a header cell in a table. A table row is denoted by the tag <tr> and is used to create a row in a table. To define a cell in a table, you use the tag <td>. When you want to group the footer content in a table, you will use the tag <tfoot> and when you want to group the header content in a table, you use the tag <thead>. Finally, when you want to create a caption in a table, you use the tag <caption>, immediately after the <table> tag.

What is the difference between colspan and rowspan?

    Colspan is what is used to make a cell span over multiple columns, where rowspan does the same thing but over multiple rows.

What is the difference between cells spacing and padding?

    The difference between cell spacing and padding is that the padding will put space between the edges of the cell and its content, whereas cell spacing is the space between each cell.

How do you define table borders using CSS? How do you control border widths using CSS?

    You use the border function within CSS to define table borders. This way you can change how the borders on the cells within your table look like. To control border widths, you will use the "border-width" property to set the width of your table cell's borders.
